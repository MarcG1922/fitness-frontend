# Fitness App

Aplicación web de fitness para registrar entrenamientos, seguimiento de peso corporal y tips de ejercicios.

## Características
- Registro e inicio de sesión con JWT.
- Dashboard con calendario semanal.
- Registro y visualización de entrenamientos.
- Seguimiento de peso corporal con gráfico de progresión.
- Sección de tips de ejercicios con ejemplos y videos.
- Autenticación protegida para todas las rutas importantes.

## Tecnologias
- Frontend: React + Vite + Axios
- Backend: Node.js + Express + MongoDB
- Hosting: Vercel (frontend) + Railway (backend)
